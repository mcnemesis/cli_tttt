HIGHLIGHTS: Apart from the (now documented) fact that unlike older languages such as popular C with about a total of 250-300 functions in just its standard library + 32 reserved keywords, the TEA language, which doesn't need non-standard functions, comes in with just 26 key letters a to z or A to Z, and a global maximum of just 624 possible distinct commands via its 4 standard command qualifiers *, !, . and @.


TEA is a general-purpose language just like C, JavaScript, LISP or Python. However, only time will tell what it is to become most useful for. As of this moment, TEA has proven capable of the following feats;

1. The shortest programs per character such as for the Hello World standard.

2. A robust and exhaustive text processing suite more advanced than any modern language we currently know.

3. The easiest instruction set to master given each of the 26 letters of the alphabet stands for a clear and distinctive utility space in the TEA language.

4.  Very idiomatic instructions and signatures; such as just the following minimal, but powerful examples;

n:1000:10 ---(numbers)-> to generate random numbers between 10 and 1000

i!:{Name?}|i:|x:{Hi } --(interact)-> to prompt for and greet someone with their name

i!:{Long input}|g: --(glue)-> to eliminate white space from any input text.

a:{one two three go} --(anagrammatize)-> to shuffle a list of words

r.:{1/2*(1 + 5(2-1))} --(rewrite)-> to reduce expressions to their mathematical values

and that's just scratching the surface! TIP of an ICEBERG!

5. TEA comes with a very user friendly learning and working environment useable from anywhere; WEB, Mobile, Online and Offline capable.

6. TEA programs developed from anywhere can be used from anywhere TEA works; embedded on websites, used to build APIs, used to power servers, scripts, standalone mobile-web apps, or integrated into desktop apps. We have built several tools already to help demonstrate and prove such usecases.

7. TEA has/can be utilized for highly specialized practical domains such as;

+ portable database-powered programs or scripts (no need to install a custom database, small as it seems, TEA comes with a ready to use database inbuilt). 

+ ability to process TEA programs embedded inside of text files, documentation or even TEA code embedded in the source code of other programming languages!

+ very straightforward and robust web client programming (with support for GET & POST HTTP verbs, Custom Headers and a very powerful content writing and reading/processing facility)

+ arbitrary power for designing and testing information security programs such as ciphers, encryption and decryption, steganography, obfuscation, advanced mathematics codes and even such peculiar things as writing using space or invisible text messages!

+ designing and running lightweight interactive text-based games of many kinds, and which can be both engaging and with simple to complex reward logic.

And so much more...

8. TEA programs are generally small and lightweight enough that many difficult solutions can be written so as to fit within a single file and not just that, but that, especially when a TEA program is minified, it usually can readily fit inside just a few characters or just a single string!
