WEB TEA IDE

This is the official WEB Integrated Development Environment for the Transforming Executable Alphabet computer programming language. You can use this interface to learn, try-out or apply the TEA language to solve actual problems by loading text input, selecting or composing a TEA program, then executing it against the provided input to see the results.
